# Aikido DAB Lern-App

Progressive Web App für Aikido-Techniken nach DAB-Prüfungsordnung.

## Features

✅ 2 Lerngänge: Techniken üben & Angriffsnamen lernen  
✅ Spaced Repetition System  
✅ Prüfungssimulation mit Auswertung  
✅ 190 Techniken aus DAB-Prüfungsordnung  
✅ Video-Links mit Zeitmarkern  
✅ Gürtelfarben-Filter (Gelb/Orange/Grün/Blau/Braun)  
✅ Hell/Dunkel-Modus  
✅ Admin-Interface für Daten-Updates  
✅ Offline-fähig  
✅ Handy-optimiert  

## Quick Start

1. `aikido-schwachpunkte.html` im Browser öffnen
2. Gürtelfarbe wählen
3. Karte antippen → Lösung sehen
4. Bewerten: Kann ich / Geht so / Schwer
5. Fertig!
